This is an initial release.

The package provides helpers to interact with an online API.
It's not appropriate to make real requests to that API during
testing or documentation compilation. As such, most examples
are \dontrun{}, however I have included a copy of the real
output. Unit tests are included that mock the API responses
by replaying real responses that I've bundled with the tests.
